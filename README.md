# Azure DevOps Directory Migration

This project describes the steps to migrate the directory of an Azure DevOps Services organization from one Microsoft Entra ID (Azure AD) tenant to another.

## Migration Plan

See [AzDO-Migration-Plan.md](AzDO-Migration-Plan.md) for the comprehensive migration plan covering:

- **Prerequisites and third-party dependencies** — Azure, Entra ID, and external tooling requirements
- **Users and groups migration in Microsoft Entra ID** — Detailed identity migration strategy
- **Prioritized migration steps** — Step-by-step plan with complexity and risk assessments
- **AI-assisted automation** — How to leverage GitHub Copilot to accelerate migration tasks
- **Migration checklist** — Complete pre/during/post-migration checklist
- **Microsoft official references** — Links to relevant Microsoft documentation
